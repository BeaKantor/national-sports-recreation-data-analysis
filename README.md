# national-sports-recreation-data-analysis
National-level data analysis of sports, recreation and entertainment activities using Tableau. Business Intelligence academic project.



##  Project Overview

This project focuses on the national-level analysis of data in the domain of sports, recreation, and entertainment activities.

The objective was to explore, analyze, and visualize key indicators in order to identify trends, patterns, and performance metrics at a national scale.

The project was developed as part of a Business Intelligence academic assignment.



##  Collaboration

This project was developed in collaboration with a colleague as a team assignment.

Both contributors were involved in:
- Data understanding and preparation
- KPI selection
- Dashboard design
- Interpretation of results
- Documentation and presentation
